## Instalasi dan Penggunaan

### Persyaratan Sistem
- Python 3.7 atau lebih baru
- Pygame library

### Langkah Instalasi

1. **Clone repository**
   ```bash
   git clone https://github.com/Renn-devBI/Fast-O-Tuan---Lirik-Commad.git
   cd Fast-O-Tuan---Lirik-Commad
   ```

2. **Install dependencies**
   ```bash
   pip install pygame
   ```

3. **Jalankan aplikasi**
   ```bash
   python app.py
   ```
